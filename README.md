# RPG Nusantara Interaktif (Next.js)

Web game RPG sederhana berbasis Next.js dengan fitur:

- Eksplorasi untuk menemukan musuh acak.
- Sistem pertarungan turn-based sederhana.
- Leveling (EXP, naik level, peningkatan ATK).
- Ekonomi mini (emas dan pembelian ramuan).
- Log pertarungan agar progres mudah dipantau.

## Menjalankan lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.
